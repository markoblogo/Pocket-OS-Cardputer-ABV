# Security Notes

- Never commit real Wi-Fi passwords.
- Never commit real AI API keys.
- Keep real `/config/wifi.json` and `/config/ai.json` only on your SD card or local ignored files.
- The Web File Manager is intended for trusted local networks only and may be unauthenticated.
- Do not expose the device to public networks or port-forward it from a router.
- Treat uploaded/downloaded SD files as untrusted content.
- Payments are unsupported. Do not store payment card data. Do not emulate payment cards.
- If sharing logs, check `/logs/system.log` for network names or endpoint URLs first.
